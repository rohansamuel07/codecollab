- 👥 **Real-Time Collaboration**  
  Live editing and broadcasting with Socket.IO and Yjs.

- 🧠 **AI Copilot + Debugger**  
  Pollinations AI and StarCoder help with debugging and code suggestions.

- 📝 **Multi-Language Support**  
  Supports 10+ languages through the Piston API.

- 🖊️ **Advanced Editor**  
  Powered by Monaco Editor with syntax highlighting, autocompletion, and themes.

- 🔐 **Room-Based Sessions**  
  Join via room ID for secure, invite-only collaboration — no login required.

- 💾 **Save & Retrieve Files**  
  Personal file management backed by MongoDB.

- 🎨 **tldraw Integration**  
  Draw, explain, and collaborate visually with a shared whiteboard.

- 📱 **Responsive Design**  
  Mobile and desktop friendly with Tailwind CSS UI.
